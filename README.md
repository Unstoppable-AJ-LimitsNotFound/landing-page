# Landing Page

A landing page built from scratch as part of 
The Odin Project's Foundations curriculum.

## About

This project involved replicating a provided 
design spec using only HTML and CSS — no 
frameworks or libraries.

## What I built

- A fixed header with logo and navigation links
- A hero section with a two-column layout
- An info section with a row of cards
- A quote/testimonial section
- A call-to-action section
- A footer

## What I used

- Semantic HTML5 elements
- CSS Flexbox for all layout and alignment
- CSS custom properties for colors and spacing
- Git for version control with high quality 
  commit messages

## What I learned

- Translating a visual design into HTML 
  structure before writing CSS
- Nesting flex containers to build 
  complex layouts
- Resetting default browser styles
- Using semantic HTML elements over generic divs
- Writing proper git commit messages

## Built with

HTML | CSS | Flexbox
